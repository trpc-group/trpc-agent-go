- Each main question belongs to one dimension.
- Main-question selected option values are 1, 2, or 3.
- drink_gate_q1 values are 1, 2, 3, or 4.
- drink_gate_q2 values are 1 or 2.
- Each dimension has exactly two main questions. Sum their numeric values.
- Convert dimension scores to levels with these thresholds: <=3 is L, 4 is M, >=5 is H.
- Build the ordered 15-dimension level sequence in the exact dimension_order provided by the reference material.
- Compare the user sequence to every normal type pattern.
- Treat L=1, M=2, H=3. Distance is the sum of absolute differences across the 15 dimensions.
- Similarity is max(0, round((1 - distance / max_distance) * 100)).
- Rank candidates by smaller distance, then larger exact matches, then larger similarity.
- exact_matches means the number of dimensions whose L/M/H level exactly matches the chosen normal candidate pattern.

- Otherwise, primary result is the best normal candidate and there is no secondary result.
